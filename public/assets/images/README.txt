Image replacement guide

You can replace every placeholder image manually with your own files.

Supported formats:
- PNG
- JPG / JPEG
- WEBP

How replacement works:
- Keep the same base file name.
- You can use any of the supported extensions above.
- Example:
  - profile-main.png
  - profile-main.jpg
  - profile-main.webp

Current base names in use:
- profile/profile-main
- profile/profile-portrait
- projects/ecommerce-platform
- projects/saas-dashboard
- projects/mobile-banking
- projects/ai-content-tool
- projects/design-system
